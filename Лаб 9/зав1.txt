1.Враховуючи попередні лабораторні роботи, створіть файл XML або JSON із довільними даними для стовпчатої діаграми. Внесіть у нього дані.
Зчитайте дані з файлу, на основі цих даних побудуйте діаграму
Додайте на діаграму підпис - Група, ПІП, дата
Змініть сценарій таким чином, щоб у якості додаткових параметрів він приймав вихідні розміри зображення.

<?php
// Створення зображення розміром 400x300 пікселів
$image = imagecreate(400, 300);

// Визначення кольорів
$bgColor = imagecolorallocate($image, 255, 255, 255);
$barColor = imagecolorallocate($image, 0, 0, 255);

// Заповнення фону
imagefilledrectangle($image, 0, 0, 400, 300, $bgColor);

// Побудова стовпчатої діаграми (це приклад, замість цього використайте ваші дані)
imagefilledrectangle($image, 50, 100, 100, 300, $barColor);
imagefilledrectangle($image, 150, 50, 200, 300, $barColor);
imagefilledrectangle($image, 250, 150, 300, 300, $barColor);

// Виведення зображення в браузер
header('Content-Type: image/png');
imagepng($image);

// Звільнення ресурсів
imagedestroy($image);
?>
